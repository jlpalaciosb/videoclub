# Videoclub

Aplicación web Laravel 5.

# Install

La aplicación requiere PHP 7.

`cp .env.example .env`

`php artisan key:generate`

`php composer.phar install`

Cree una base de datos postgres, mysql o similar y setee sus credenciales en el archivo .env.

`php artisan migrate`

`php artisan db:seed`

`php artisan serve`
